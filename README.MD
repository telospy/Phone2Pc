## ✨ Features

- 📱 **Phone to PC** - Upload files from your phone directly to computer
- 💻 **PC to Phone** - Download files from PC to phone with one tap
- 🔄 **PC to PC** - Share files between computers
- 📸 **QR Code Access** - Instant connection with phone camera
- 📁 **Folder Upload** - Upload entire folders (auto-compressed to ZIP)
- 👁️ **File Preview** - View images, PDFs, and text files
- 💬 **Send Messages** - Send text messages between devices
- 🔒 **Secure** - End-to-end encrypted, files stay on your PC

## 🚀 Quick Start

### For Users (No installation needed!)

1. Visit **[https://phone2pc.onrender.com](https://phone2pc.onrender.com)**
2. Click "Create Server"
3. Enter your free [ngrok token](https://dashboard.ngrok.com/signup)
4. Select a folder to save files
5. Click "Start Server"
6. Scan QR code with your phone
7. Start transferring files!
